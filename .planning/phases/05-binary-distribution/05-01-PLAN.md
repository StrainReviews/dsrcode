6. Update DefaultConfigPath() with copy-on-first-find migration per DIST-30:
   ```go
   func DefaultConfigPath() string {
       home, err := os.UserHomeDir()
       if err != nil {
           return "dsrcode-config.json"
       }
       newPath := filepath.Join(home, ".claude", "dsrcode-config.json")
       if _, err := os.Stat(newPath); err == nil {
           return newPath
       }
       // Backward compat: if old name exists, copy to new name and return new path.
       // Old file remains as backup per DIST-30. 6-month support per DIST-31.
       oldPath := filepath.Join(home, ".claude", "discord-presence-config.json")
       if _, err := os.Stat(oldPath); err == nil {
           data, readErr := os.ReadFile(oldPath)
           if readErr == nil {
               if writeErr := os.WriteFile(newPath, data, 0644); writeErr == nil {
                   slog.Warn("migrated config to new name; old file kept as backup",
                       "old", oldPath, "new", newPath)
                   return newPath
               }
           }
           // Copy failed: fall back to old path so the user is not locked out
           slog.Warn("config migration failed, using legacy path",
               "old", oldPath, "new", newPath)
           return oldPath
       }
       return newPath // Default to new name for fresh installs
   }
   ```
   Add `"log/slog"` to imports if not already present.

   Apply the same copy-on-first-find pattern to ALL other path functions that fall back to old names:

   **DefaultLogPath() / defaultLogFile():**

   ```go
   func defaultLogFile() string {

       if err != nil {
           return "dsrcode.log"

       return filepath.Join(home, ".claude", "dsrcode.log")
   }
   ```
   (Log file is append-only; no migration copy needed — new sessions write to new path.)

   **DefaultPidPath() — if this function exists:**
   Apply copy-on-first-find if it resolves a PID file path; otherwise just rename to dsrcode.pid.

   The key rule: any function that previously returned a `discord-presence-*` path and has a fallback MUST follow the copy-on-first-find pattern: read old → write new → log warning → return new path. If write fails, return old path (fail safe, not fail hard).
