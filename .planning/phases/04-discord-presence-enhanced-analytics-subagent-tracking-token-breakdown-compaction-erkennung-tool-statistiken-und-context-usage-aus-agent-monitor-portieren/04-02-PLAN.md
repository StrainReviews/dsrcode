Create analytics/tracker.go with `package analytics`.

```go
type Tracker struct {
    mu       sync.RWMutex
    sessions map[string]*SessionAnalytics
    enabled  bool // D-34: feature toggle
}
```

Methods:
- `func NewTracker(enabled bool) *Tracker` — initialize with empty map, enabled flag
- `func (t *Tracker) RecordTool(sessionID, toolName string)` — lock, ensure session exists (lazy init per D-21), call RecordTool on session's Tools map, persist via SaveTools, unlock. Must be O(1) in-memory (D-20). Persist call is the only IO and is non-blocking to the hook handler when called from a goroutine — but per D-31 write-on-event means we do write synchronously here since file IO is ~1ms on SSD.
- `func (t *Tracker) RecordSubagentSpawn(sessionID, id, agentType, description, parentID, prompt string)` — lock, SpawnSubagent, append to session's Subagents, persist via SaveSubagents, unlock
- `func (t *Tracker) RecordSubagentComplete(sessionID string, event SubagentStopEvent)` — lock, MatchSubagent on session's Subagents, if match found CompleteSubagent, persist, unlock
- `func (t *Tracker) UpdateTokens(sessionID, model string, current TokenBreakdown)` — lock, check if session has stored tokens for this model, call UpdateBaseline if current < stored, update session's Tokens[model], recalculate cost via CalculateSessionCost, persist via SaveTokens, unlock
- `func (t *Tracker) RecordCompaction(sessionID string)` — lock, increment session's Compactions, unlock (no separate persist — compaction count is derived from baselines)
- `func (t *Tracker) UpdateContextUsage(sessionID string, pct float64)` — lock, set session's ContextPct, unlock (no persist — ephemeral, comes from statusline file)
- `func (t *Tracker) GetSessionAnalytics(sessionID string) *SessionAnalytics` — RLock, deep copy, RUnlock. Returns nil if not found.
- `func (t *Tracker) LoadSession(sessionID string) error` — load persisted data from all 3 file types (tokens, tools, subagents), populate session entry. Called at startup for existing sessions.
- `func (t *Tracker) ensureSession(sessionID string)` — create entry with empty maps if not exists. Called under write lock.

All Record* methods check `if !t.enabled { return }` at the top per D-34.

Import sync, the other analytics files (same package so no import needed), and persist functions.
